# ლოჯისტიკური პორტალის არქიტექტურული სტრუქტურა

## 1. ტექნოლოგიური სტეკი
*   **ბექენდი:** Django (Python). უზრუნველყოფს ათასობით მომხმარებლის (ოთახის ნომრების), ამანათების ტრანსპორტირების ისტორიისა და ინვოისების უსაფრთხო მართვას.
*   **რეალურ დროში სერვისები:** Firebase.
    *   *Live Tracking:* კურიერის ან კონტეინერის ლოკაციის (GPS) რეალურ დროში ჩვენება.
    *   *Live Chat:* მომხმარებლისა და მხარდაჭერის გუნდის მყისიერი კომუნიკაცია.
*   **ფრონტენდი:** Pure HTML, CSS, Vanilla JS. "Instructional AR" ვიზუალის მაღალი წარმადობით (Performance) უზრუნველსაყოფად WebGL/CSS3D ტრანსფორმაციების გამოყენებით.

## 2. მონაცემთა ბაზის მოდელები (Django)
*   `UserRoom`: მომხმარებლის უნიკალური იდენტიფიკატორი (მაგ: GEO-10293), რომელიც აკავშირებს მომხმარებელს მის პირად კაბინეტთან და საფოსტო მისამართებთან.
*   `Package`: ინახავს ამანათის თრექინგ კოდს, წონას, გაბარიტებს, მიმდინარე სტატუსს (საწყობშია, გზაშია, ჩამოვიდა) და მიბმულია `UserRoom`-ზე.
*   `Invoice`: ინვოისის მოდელი, სადაც ინახება ტრანსპორტირების ღირებულება, გადახდის სტატუსი (Paid/Unpaid) და ინვოისის PDF ბმული.

## 3. ამანათების დეკლარირების მოდული
*   **მომხმარებლის ნაკადი (User Flow):** 
    1. მომხმარებელი კაბინეტში ამატებს თრექინგ კოდს და ირჩევს ონლაინ მაღაზიას.
    2. ტვირთავს ინვოისს ფასის დასადასტურებლად.
    3. ავსებს ნივთის დასახელებას.
*   **საბაჟო სინქრონიზაცია:** თუ ამანათის ღირებულება აღემატება 300 ლარს, სისტემა ავტომატურად ცვლის სტატუსს "ექვემდებარება განბაჟებას" და აგენერირებს სპეციალურ ექსპორტს RS.ge-სთვის (შემოსავლების სამსახურის API-სთან დაკავშირება).

## 4. ონლაინ გადახდის უსაფრთხოების პროტოკოლები
*   **Payment Gateway:** TBC/BOG ან Stripe ინტეგრაცია.
*   **უსაფრთხოება:** 
    *   Django-ს სერვერზე არასდროს ინახება საკრედიტო ბარათის მონაცემები (PCI-DSS შესაბამისობა).
    *   გამოიყენება Tokenization: ბანკი აბრუნებს მხოლოდ ტოკენს, რომლითაც სრულდება გადახდა `Invoice` მოდელზე.
    *   ყველა ტრანზაქცია ხორციელდება HTTPS/TLS 1.3 დაშიფრული არხით.
    *   გადახდის დასადასტურებლად ბექენდზე გამოიყენება ბანკის მიერ მინიჭებული Webhook-ების შემოწმება (HMAC Signature verification).
